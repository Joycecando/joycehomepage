# 需求文档

## 1. 应用概述

### 1.1 应用名称

李瞳个人主页

### 1.2 应用描述

一个简约清新的个人主页，展示李瞳（Joyce Lee）的基本信息与成就，并提供数字分身聊天功能，供朋友和面试官了解与互动。

## 2. 用户与使用场景

### 2.1 目标用户

- 朋友
- 面试官

### 2.2 核心使用场景

- 访客浏览个人信息，快速了解李瞳的背景与成就
- 访客通过数字分身聊天区提问，获取更多信息

## 3. 页面结构与功能说明

### 3.1 页面结构

```
个人主页
├── 头像区
├── 名字与一句话介绍区
├── 个人信息展示区
└── 数字分身聊天区
```

数字分身需基于以下信息回答问题：

- 职业或身份：英语专业大学三年级学生

## 4. 业务规则与逻辑

### 4.1 数字分身回复规则

- 数字分身需根据李瞳的背景信息生成回复
- 回复风格需友善温和，体现天马行空的想象力
- 回复内容需围绕职业身份、近期活动、擅长方向展开

### 4.2 视觉风格规则

- 整体风格：简约、清新，带有梦幻感的渐变效果
- 主色：#FFF5EE
- 辅助色1：#FFB6C1
- 辅助色2：#D4E1F1
- 信息层次清晰，避免花哨设计
- 适配手机端显示

## 5. 异常与边界情况

| 场景                       | 处理方式                           |
| -------------------------- | ---------------------------------- |
| 访客未输入问题直接点击发送 | 提示请输入问题                     |
| 数字分身无法理解问题       | 回复友好提示，引导访客换个方式提问 |
| 聊天记录过长               | 保持可滚动查看历史对话             |
